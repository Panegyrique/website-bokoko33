Automated drug dispenser | 2018

Project to improve the medication process, especially for patients receiving home care. The goal is to reduce the workload of caregivers, who often have to travel to dispense medications, allowing them to focus on more urgent care. The dispenser is secure, with access granted only to caregivers who can easily refill the dispenser for several weeks and configure the dispensing schedule using an intuitive human-machine interface. I was responsible for the coding and all the electronics; I based the project on Arduino, which is C++ based. Head Fighter | 2019

2D fighting game coded in Python that can be played by two players on the same computer. In this game, players can choose from a range of unique fighters, each represented only by their head, hence the name. The objective of the game is to decrease the opponent's life points by jumping on them. The game features intuitive controls and vibrant graphics that create an immersive and entertaining experience for players.

Connected hive | 2019

A project to help beekeepers, a profession that can often be difficult. This connected hive, accessed via an app developed on Android Studio, allows beekeepers to monitor the weight of honey in the hive, humidity levels, bee flow, and even detect possible theft or swarming. Access to the inside of the hive is only possible by unlocking via the app, which uses Bluetooth for unlocking. The electronic components are based on Arduino and the application and Arduino interact through a MySql database, the requests being sent in PHP.

Website for the Abeille Des Aydes Basket of Blois | 2020

The website was developed for the basketball club of Blois in order to improve the follow-up of the players, especially their youngsters of the espoir team. The Blois basketball club is a professional club playing in the Betclic Elite league (formerly Pro A). Players can record their daily experiences, such as the level of difficulty of training, their level of fatigue beforehand, the quality of their sleep ... Coaches have access to this data and can have a personalized follow-up for each player, allowing them to better target their training. They also have access to the technical characteristics of each player. Parents of young players (but also of older ones ;) ) can also consult some results. To enter the data, players or coaches simply need to log into the website, which is based on HTML, CSS, PHP and MySQL.

Tarot game | 2021

Creation of a C program to play a traditional game of tarot. Management of the code in several blocks: initialization of the game, unfolding of the tricks and display of the score. Development of a graphical interface using the SDL library.

Side channel attacks on embedded devices using standard radio equipment | 2023 - 2024

This project focuses on exploring vulnerabilities in embedded systems by demonstrating that side channel attacks can be successfully carried out using standard radio equipment and a limited budget. The aim is to demonstrate that modest financial resources and accessible technologies can be sufficient to highlight vulnerabilities, to raise awareness of the reality of security threats, and to promote cost-effective approaches to enhancing the security of embedded systems.This project focuses on exploring vulnerabilities in embedded systems by demonstrating that side channel attacks can be successfully carried out using standard radio equipment and a limited budget.
